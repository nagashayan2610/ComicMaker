---# ComicMaker #---

An AI-powered comic generator built with Python and Stable Diffusion Turbo that converts screenplay-style scripts into comic book pages automatically.


---# Overview #---

ComicMaker reads a screenplay-style text file, extracts scenes, characters, and dialogue, generates AI illustrations for every dialogue, places speech bubbles on each panel, and
finally combines multiple panels into complete comic pages.

The project demonstrates AI image generation, natural language parsing, image processing, and automated comic layout generation.


---# Features #---

-  Parse screenplay-style scripts
-  AI-generated comic panels using Stable Diffusion Turbo
-  Automatic speech bubble generation
-  Automatic comic page creation
-  Saves comics as PNG images
-  Supports multi-page comics
-  Fast image generation using SD-Turbo


---# Technologies Used #---

- Python
- Diffusers
- Stable Diffusion Turbo
- PyTorch
- Pillow (PIL)
- Regular Expressions (re)


---# Project Structure #---

>>> src/ <<<
Contains the core source code responsible for the comic generation pipeline.

- >> project.py <<
  - Entry point of the application.
  - Controls the complete workflow from reading the script to generating the final comic pages.
  - Organizes dialogues into pages and saves the generated output.

- >> parser.py <<
  - Reads the screenplay-style text file.
  - Identifies scene headings, character names, and dialogues.
  - Converts the script into structured data for further processing.

- >> ai_generator.py <<
  - Loads the Stable Diffusion Turbo model.
  - Creates AI-generated comic panel illustrations from scene descriptions and dialogue.
  - Uses prompt engineering to produce comic-style artwork.

- >> layout.py <<
  - Designs each comic panel.
  - Draws speech bubbles.
  - Places character names and dialogue onto the generated images.

- >> page_builder.py <<
  - Combines multiple comic panels into a single comic page.
  - Dynamically adjusts the page height according to the number of panels.
  - Saves the completed page as a PNG image.

>>> scripts/ <<<

Contains screenplay input files.

- >> scene.txt <<
  - Stores the screenplay in a screenplay-style format.
  - Acts as the primary input for comic generation.

>>> sample_output/ <<<

Contains sample comic pages generated by ComicMaker.

These images demonstrate the expected output of the project and provide a quick preview of the generated comics.

 >>> output/ <<<

Stores newly generated comic pages after each execution.

>>> backup/ <<<

These images can be backup if we like it as our wish.

---# Usage #---

Place your screenplay inside:

Run:

python src/project.py

Enter a comic name when prompted.

The generated comic pages will be saved inside the output folder.


---# Script Format #---

Rules:

1. SCENE FORMAT
INT. or EXT. must start scene lines

Example:
INT. ROOFTOP - NIGHT

2. CHARACTER FORMAT
Must be uppercase only

after character is dialogue

Example:
I will save.

3. DIALOGUE FORMAT
Next line

4. COMMENT BLOCK (IGNORED BY SYSTEM)
Anything inside triple quotes is ignored:

5. DO NOT USE:
- colon after character name
- free paragraph writing
- mixed formats

6. RECOMMENDED STYLE:
